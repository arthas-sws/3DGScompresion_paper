# Attribution

This repository packages Codex Skills for 3D Gaussian Splatting paper workflows.

The scripts use public metadata and PDFs supplied by their original publishers,
including arXiv when the user requests arXiv retrieval. Paper PDFs, titles,
abstracts, project pages, and code repositories remain copyrighted or licensed
by their respective authors and publishers.

The Skill instructions and utility scripts in this repository are released under
the repository license unless a file states otherwise.

The `innovation-review` mode is repository-authored workflow guidance for
reviewing user-provided papers. It does not redistribute third-party PDFs or
claim ownership over paper tables, figures, project pages, or code.
