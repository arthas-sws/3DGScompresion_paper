# Output Profiles

默认使用“中文精读”。用户明确要求时可切换。

## 中文精读

完整保留方法、实验、结果、效率、消融、局限和复现结论。适合复现调研和综述。

## 审稿模式

增加：

- 主要优点；
- 主要问题；
- 对作者的问题；
- 可能拒稿/接收依据；
- 实验补充建议。

仍必须输出 JSON。

## 复现模式

增加：

- 环境依赖；
- 数据准备；
- 训练、渲染、评估命令；
- 论文结果到代码命令的映射；
- 复现风险。

仍必须输出 JSON。

## 快速速读

可以压缩篇幅，但不能删除：

- 核心贡献；
- 主要结果；
- 代价；
- 局限；
- 未报告项；
- JSON 结构化结果。
