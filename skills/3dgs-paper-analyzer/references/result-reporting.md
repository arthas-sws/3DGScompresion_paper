# Result Reporting

## Markdown 必备结果表

```markdown
| 数据集 | 场景 | 指标 | 本文方法 | baseline | 差值 | 趋势 | 可比性 | 证据 | 说明 |
|---|---|---:|---:|---:|---:|---|---|---|---|
```

每行必须能映射到 JSON 的 `analysis.main_results`。

## 效率与代价

单独汇总：

- 渲染 FPS 或延迟；
- 训练时间；
- 显存；
- 模型大小；
- Gaussian 数量；
- 编码、解码或预处理代价。

未报告项显式写“论文未报告”。

## 消融

每个消融项记录：

- 组件名称；
- 移除或替换方式；
- 影响的指标；
- 证据位置；
- 该实验是否真正隔离变量。

## 结论边界

结果分析必须回答：

- 实验证明了什么；
- 尚未证明什么；
- 提升是否稳定；
- 代价是什么；
- 哪些比较不可直接比较。
